# 电商开放平台接口开发SDK-专为简化电商系统开发而生

## 一、项目概述

电商开放平台接口开发SDK-为简化电商ERP系统开发而生，集合了各电商开放平台API接口开，已经支持的开放平台包括：淘宝天猫、京东POP、京东自营、拼多多、抖店、微信小店等。

SDK支持输出各平台原生字段数据和封装之后的标准字段数据，可自行选择使用。使用SDK开发可以让您专注自己的业务，复杂繁琐的数据交给SDK就可以了，极大节省开发者系统开发时间。
